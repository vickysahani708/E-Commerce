# MERN E-Commerce Project

This is a **full-stack E-Commerce application** built using the MERN stack (MongoDB, Express, React, Node.js). The project includes:

- **Admin Dashboard**: Manage products, users, orders, and analytics.  
- **Client Frontend**: Users can browse products, add to cart, place orders, and manage their account.  
- **Server / Backend**: REST API for authentication, product management, and order processing.

## Features

### Client
- Browse products by category
- Search and filter products
- User registration and login
- Add/remove products to/from cart
- Place orders and track status

### Admin
- Add, update, delete products
- View and manage users
- Manage orders
- Dashboard analytics

### Server
- REST API endpoints for client and admin
- JWT-based authentication and authorization
- Error handling and data validation

## Installation

### Prerequisites
- Node.js (v14 or above)
- MongoDB (local or Atlas)
- npm / yarn

### Steps
1. **Clone the repo**
```bash
git clone https://github.com/vickysahani708/E-Commerce.git
cd E-Commerce
# Backend
cd server
npm install

# Frontend
cd client
npm install
# Admin
cd admin
npm install
```
